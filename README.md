# 📐 수학 시험지 에디터 (Math Exam Editor)

**LLM(AI)과 협업하여 고품질의 수학 교재를 만드는 웹 기반 에디터입니다.** 설치 없이 브라우저에서 바로 실행되며, 로컬 폴더와 연동되어 이미지와 데이터를 안전하게 관리합니다.

🔗 **접속 링크:** [https://sykaboom.github.io/pdf-test-builder/](https://sykaboom.github.io/pdf-test-builder/)

---

## ✨ 핵심 기능

* **⚡️ 설치 없는 실행:** 크롬(Chrome), 엣지(Edge) 등 최신 브라우저에서 링크 클릭만으로 작동합니다.
* **📂 로컬 폴더 연동:** `File System Access API`를 사용하여 내 컴퓨터의 폴더에 이미지와 데이터를 직접 저장합니다.
* **∑ 수식 최적화 (Atomic Math):** MathJax 기반의 수식 렌더링을 지원하며, 수식을 더블 클릭하여 소스 코드를 수정할 수 있습니다.
* **🖼️ 이미지 자동 관리:** 붙여넣기(`Ctrl+V`) 시 이미지가 자동으로 지정된 로컬 폴더에 저장됩니다.
* **🤖 LLM 친화적:** AI가 생성한 텍스트 포맷(`[[스타일_라벨]]`)을 자동으로 인식하여 레이아웃을 구성합니다.

---

## 🚀 시작하기 (Getting Started)

### 1. 작업 폴더 준비
내 컴퓨터에 작업할 빈 폴더를 하나 만듭니다. (예: `[수학작업]`)

### 2. 에디터 접속 및 연동
1.  [에디터 페이지](https://sykaboom.github.io/pdf-test-builder/)에 접속합니다.
2.  왼쪽 사이드바의 **[📂 작업 폴더 열기]** 버튼을 클릭합니다.
3.  준비한 폴더를 선택하고, 브라우저 상단의 **'파일 보기 및 저장 권한' 요청을 [허용]**합니다.
